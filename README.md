# mydearbirthday

В классе Solution реализуй 4 метода:

метод nowExample должен вернуть текущую дату;
остальные методы должны вернуть дату 12 сентября 2020 года, но разными способами.



Псевдокод:

```
ПРОГРАММА "Освоение нового API"

ОСНОВНАЯ ФУНКЦИЯ:
    ВЫВЕСТИ результат функции nowExample()
    ВЫВЕСТИ результат функции ofExample()
    ВЫВЕСТИ результат функции ofYearDayExample()
    ВЫВЕСТИ результат функции ofEpochDayExample()

ФУНКЦИЯ nowExample():
    ВОЗВРАТ текущую дату системы

ФУНКЦИЯ ofExample():
    ВОЗВРАТ дату: 12 сентября 2020 года

ФУНКЦИЯ ofYearDayExample():
    ВОЗВРАТ дату, соответствующую 256-му дню 2020 года

ФУНКЦИЯ ofEpochDayExample():
    ВОЗВРАТ дату, соответствующую 18517-му дню от начала эпохи (1970-01-01)
```

Пояснения:

· nowExample() - получает текущую дату компьютера
· ofExample() - создает конкретную дату (год, месяц, число)
· ofYearDayExample() - создает дату по номеру года и дню в этом году
· ofEpochDayExample() - создает дату по количеству дней от 1 января 1970 года (Unix epoch)

Каждая функция возвращает объект даты, который затем выводится на экран в основной программе.
